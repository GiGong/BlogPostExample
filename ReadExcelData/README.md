# Read excel data with C#
[Blog Post](https://www.gigong.io/2022/02/07/CSharp-read-excel-data)

Excel 데이터를 읽어오는 C# 콘솔 프로젝트 입니다.

입력은 두가지가 필요하며  
첫번째는 엑셀 파일 경로, 두번째는 읽어올 엑셀 파일의 column 수 입니다.

읽어온 데이터를 tab 간격으로 출력하고 종료합니다.
